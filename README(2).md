# Сессия МИИТа — версия для GitHub Pages

## Структура

- `index.html` — интерфейс и игровая логика;
- `questions.json` — отдельная база вопросов;
- `.nojekyll` — отключает обработку Jekyll на GitHub Pages.

## Логика сессии

В игре есть 6 преподавателей:

1. Безряков;
2. Королёва;
3. Ефимов;
4. Данилина;
5. Рыжков;
6. Черепанов.

При каждом запуске новой сессии игра случайно выбирает **ровно 4 преподавателей из 6**. Порядок выбранных преподавателей также случайный.

Вопросы Данилиной, Королёвой и Ефимова находятся в отдельной расширенной базе. Для Безрякова, Рыжкова и Черепанова используются точные исходные вопросы из первоначального HTML — формулировки, варианты ответов и правильные ответы не изменялись.

Показанные вопросы сохраняются в `localStorage` браузера и не повторяются, пока не закончится очередь вопросов соответствующего преподавателя. Варианты ответов перемешиваются отдельно.

## Публикация

Загрузите в корень репозитория все четыре файла:

```text
index.html
questions.json
README.md
.nojekyll
```

В GitHub откройте **Settings → Pages**, выберите публикацию из ветки `main` и папки `/ (root)`.

После обновления сайта выполните жёсткую перезагрузку страницы: `Ctrl + F5` или `Cmd + Shift + R`.

## Источники банков

- Данилина, Королёва, Ефимов — новые расширенные экзаменационные банки;
- Безряков, Рыжков, Черепанов — точные исходные банки из первой версии HTML.
