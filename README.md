# Сессия МИИТа — GitHub Pages

Проект состоит из двух основных файлов:

- `index.html` — интерфейс и игровая логика;
- `questions.json` — отдельная база из 170 вопросов.

## Состав базы

- Данилина — 74 вопроса по экономике предприятия;
- Королёва — 66 вопросов по основам комплексной безопасности;
- Ефимов — 30 вопросов по методам оптимизации.

## Как выложить на GitHub Pages

1. Загрузите **в корень репозитория** файлы `index.html` и `questions.json`.
2. Не помещайте `questions.json` внутрь отдельной папки: он должен лежать рядом с `index.html`.
3. В GitHub откройте `Settings → Pages`.
4. Выберите `Deploy from a branch`, ветку `main`, папку `/root` и сохраните.
5. После обновления GitHub Pages выполните жёсткое обновление страницы: `Ctrl+F5` или `Cmd+Shift+R`.

## Как работает смена вопросов

Вопросы выбираются через shuffle-bag и сохраняются в `localStorage`. Уже показанные вопросы не возвращаются, пока не будет пройдена основная часть базы конкретного преподавателя. Варианты ответов также перемешиваются при каждом показе.

После изменения базы увеличьте поле `version` в `questions.json`. Тогда сохранённая очередь вопросов автоматически сбросится.

## Локальный запуск

Из-за ограничений браузера `questions.json` может не загрузиться при открытии `index.html` двойным кликом. Для локальной проверки запустите в папке проекта:

```bash
python3 -m http.server 8000
```

Затем откройте `http://localhost:8000`.
